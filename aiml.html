Introduction to AI

What is Artificial Intelligence (AI)?

Artificial Intelligence (AI) is the science and engineering of creating intelligent machines that can perform tasks that typically require human intelligence. These tasks include understanding language, recognizing patterns, solving problems, and making decisions. AI is all around us—from virtual assistants like Siri and Alexa, to self-driving cars, to recommendation systems on Netflix and Amazon.


What is Machine Learning (ML)?

Machine Learning (ML) is a subset of AI that focuses on teaching machines to learn from data and improve their performance over time without being explicitly programmed. Instead of following rigid rules, ML systems analyze patterns in data and make predictions or decisions based on what they learn.

Applications: Self-driving cars, speech recognition, recommendation systems, medical diagnosis.

Supervised vs Unsupervised Learning

What is Supervised Learning?

Supervised Learning is a type of Machine Learning where the model learns from labeled data. Labeled data means each example comes with the correct answer. The model learns a mapping between input data and output labels so it can predict the correct output for new data.

Analogy: It’s like a student learning with a teacher. The teacher provides questions (inputs) and correct answers (labels). Over time, the student learns to answer new questions correctly.

How It Works (Step by Step)
  1. Collect Data: Gather examples with inputs and corresponding outputs. Example: Dataset of houses with features (size, location) and prices (labels).
  2. Split Data: Divide data into training (to teach the model) and testing (to check performance).
  3. Train the Model: Feed the training data to an algorithm. The model learns the relationship between inputs and outputs.
  4. Make Predictions: Apply the trained model to new data to predict outputs.
  5. Evaluate Performance: Measure accuracy or error using metrics like Mean Squared Error (regression) or Accuracy/F1-score (classification).
Types of Supervised Learning
  • Regression: Predicts a continuous value. Example: Predicting house prices, temperature, or salary.
  • Classification: Predicts a discrete label or category. Example: Email spam detection (spam or not spam), predicting if a tumor is malignant or benign.
Popular Supervised Learning Algorithms
  • Linear Regression – predicts continuous values
  • Logistic Regression – predicts probabilities for classification
  • Decision Trees / Random Forest – works for both regression & classification
  • Support Vector Machines (SVM) – separates data into classes
  • K-Nearest Neighbors (KNN) – predicts based on closest examples
Key Points to Remember
  • Requires labeled data
  • Learns patterns from input-output pairs
  • Useful for prediction and classification tasks
  • Model performance is measured on unseen test data

The model is trained on labeled data, meaning the input comes with the correct output.


# Linear Regression Example
from sklearn.linear_model import LinearRegression

X = [[1], [2], [3], [4]]
y = [2, 4, 6, 8]
model = LinearRegression()
model.fit(X, y)
print(model.predict([[5]]))  # Output: 10

What is Unsupervised Learning?

Unsupervised Learning is a type of Machine Learning where the model learns from unlabeled data. Unlike supervised learning, there are no correct answers provided. The model tries to find patterns, relationships, or structure in the data on its own.

Analogy: It’s like exploring a new city without a map. You try to group similar places together or find patterns in how things are arranged.

How It Works (Step by Step)
  1. Collect Data: Gather examples without labels. Example: Customer data with attributes like age, spending, and location.
  2. Choose an Algorithm: Select an unsupervised algorithm such as clustering or dimensionality reduction.
  3. Train the Model: Feed the data to the algorithm. The model identifies patterns or groups within the data.
  4. Analyze Results: Interpret the clusters, associations, or patterns discovered by the model.
Types of Unsupervised Learning
  • Clustering: Groups data points based on similarity. Example: Customer segmentation, grouping articles by topic.
  • Dimensionality Reduction: Reduces the number of features while keeping important information. Example: Simplifying data for visualization or speeding up computation.
Popular Unsupervised Learning Algorithms
  • K-Means Clustering – groups data into clusters based on similarity
  • Hierarchical Clustering – creates a hierarchy of clusters
  • DBSCAN – density-based clustering algorithm
  • PCA (Principal Component Analysis) – reduces dimensions while preserving variance
  • t-SNE – visualizes high-dimensional data in 2D or 3D
Key Points to Remember
  • Works with unlabeled data
  • Finds hidden patterns or structures in the data
  • Useful for clustering, association, and dimensionality reduction
  • Helps in data exploration and understanding relationships

# K-Means Clustering Example
from sklearn.cluster import KMeans
X = [[1,2], [1,4], [1,0], [10,2], [10,4], [10,0]]
kmeans = KMeans(n_clusters=2)
kmeans.fit(X)
print(kmeans.labels_)

Reinforcement Learning (RL)

What is Reinforcement Learning?

Reinforcement Learning is a type of Machine Learning where an agent learns to make decisions by interacting with an environment. Instead of being given labeled data, the agent receives feedback in the form of rewards or penalties based on its actions.

Analogy: Think of teaching a dog new tricks. The dog tries actions, and you give it a treat (reward) when it performs correctly or ignore/penalize it for wrong actions. Over time, the dog learns the best actions to maximize treats.

How It Works (Step by Step)
  1. Agent: The learner or decision-maker (e.g., a robot, AI software).
  2. Environment: Everything the agent interacts with (e.g., a game board, maze, or physical world).
  3. Actions: Choices the agent can make at each step.
  4. Rewards: Feedback the agent receives after taking an action. Positive rewards encourage good behavior; negative rewards discourage bad behavior.
  5. Policy: The strategy the agent uses to decide actions based on the current state.
  6. Learning: The agent repeats actions, explores the environment, and adjusts its policy to maximize cumulative rewards.
Key Concepts in Reinforcement Learning
  • Exploration vs Exploitation: Exploration means trying new actions to discover better rewards, while exploitation means using known actions that give good rewards.
  • State: Current situation of the agent in the environment.
  • Action: Any move or decision the agent can take in a state.
  • Reward: Immediate feedback received after an action.
  • Episode: A complete sequence of actions from start to finish in the environment.
Applications of Reinforcement Learning
  • Training AI to play games (Chess, Go, Atari games)
  • Robotics – teaching robots to walk or pick objects
  • Self-driving cars – learning to navigate safely
  • Resource management – optimizing energy, traffic, or inventory systems
  • Recommendation systems – learning user preferences over time
Popular Reinforcement Learning Algorithms
  • Q-Learning – learns the value of actions in each state
  • Deep Q-Networks (DQN) – uses neural networks for complex environments
  • Policy Gradient Methods – directly optimizes the policy
  • Actor-Critic Methods – combines value and policy optimization
Key Points to Remember
  • Works with feedback (rewards/penalties) rather than labeled data
  • Focuses on learning optimal actions to maximize cumulative rewards
  • Involves continuous interaction with the environment
  • Widely used in gaming, robotics, autonomous systems, and recommendation engines

Neural Networks

Neural networks are a series of algorithms that recognize relationships in a dataset through layers that mimic the human brain.


# Simple ANN in Keras
from keras.models import Sequential
from keras.layers import Dense

model = Sequential()
model.add(Dense(8, input_dim=4, activation='relu'))
model.add(Dense(3, activation='softmax'))
model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])

Deep Learning

Deep Learning is a subset of ML that uses multi-layered neural networks to model complex patterns. Useful in image and speech recognition.
